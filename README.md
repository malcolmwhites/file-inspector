# File Inspector

A C++ project for reading and inspecting binary file formats.

## Current Features

- Parses WAV file headers
- Reads:
  - Audio format
  - Channels
  - Sample rate
  - Byte rate
  - Block align
  - Bits per sample
  - Data size
  - Duration

## Built With

- C++
- std::ifstream
- Binary file parsing

## Future Goals

- Read audio samples
- Display waveform information
- Compute audio statistics
- Add audio effects
